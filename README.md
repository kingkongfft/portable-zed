# Portable Zed IDE

A portable installation of the [Zed](https://zed.dev) code editor.

> **Disclaimer:** This repository is for **educational and learning purposes only**. All rights belong to Zed Industries. If you find Zed useful, please support the official project at [zed.dev](https://zed.dev).

## Usage

1. Download `Zed.zip` from the [Releases page](https://github.com/kingkongfft/portable-zed/releases)
2. Extract to any location
3. Run `zed.exe` (no installation required)
